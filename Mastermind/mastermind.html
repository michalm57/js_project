The object of MASTERMIND (r) is to guess a secret code consisting of a series of 5 colored pegs. Each guest results in feedback narrowing down the possibilities of the code. The winner is the player who solves his opponent's secret code with fewer guesses.

==> You guessed the correct color in the correct place.

==> You guessed the correct color, but it is in a different place.

==> The guessed color does not appear on the board.

==> Undo previous move.

Which color to put in the place marked with an arrow?
